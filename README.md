# The business process of warehouse management
## 1.Описание проекта
Данный проект представляет собой систему вывода процента продажи товара на складе. Она включает Telegram-бота для ввода и вывода информации и Dashboard на базе Python Dash, который визуализирует показатели склада.

Цель проекта - упростить проверку наличия товара и информации о нем, для своевременного пополнения склада.
# 
## 2.Используемые технологии
- PyCharm
- Telegram
- Dash 
- GitHub

## 5. Запуск проекта
Запустите проект из корневой директории

При успешной загрузке будут инициализированы:

- Telegram-бот, готовый принимать команды.
- Dash-дашборд.
### Инструкция по использованию
После запуска бота пользователь может выполнять основные операции напрямую через Telegram.

Бот поддерживает следующие команды:
- /dash
- /count

При команде /dash вам нужно будет скачать файл dashbord.py из папки dash и ввести в Телеграм-бота.

Бот выведет дашборд.

При команде /count вам нужно будет скачать файл product.xlsx из папки data или другую аналогичную таблицу и ввести в Телеграм-бота.

Бот выведет процентное соотношение.
