# The business process of warehouse management
## 1.Описание проекта
Данный проект представляет собой систему вывода информации о товаре на складе. Она включает Telegram-бота для вывода информации и Dashboard на базе Python Dash, который визуализирует показатели склада.

Цель проекта - упростить проверку наличия товара и информации о нем, для своевременного пополнения склада.
#
## 2.Используемые технологии
- PyCharm
- Telegram
- Dash 
- GitHub
