# The business process of warehouse management
## 1.Описание проекта
Данный проект представляет собой систему автоматизации финансовых процессов склада.

Она включает Telegram-бота для ввода ключевых финансовых данных (доходы, расходы, количество проданных товаров) и аналитический Dashboard на базе Python Dash, который визуализирует показатели в реальном времени.

Цель проекта — упростить процесс учёта, повысить точность данных и предоставить удобные инструменты для анализа финансовой деятельности образовательного центра.

## 5. Запуск проекта
Запустите проект из корневой директории

При успешной загрузке будут инициализированы:

- Соединение с базой данных и репозитории.
- Dash-дашборд.
- Telegram-бот, готовый принимать команды.
### Инструкция по использованию
Проект имеет два основных интерфейса: Telegram-бот и Дашборд.

После запуска бота пользователь может выполнять основные операции по учёту финансовых показателей напрямую через Telegram.

Бот поддерживает следующие команды:
- Добавить доход:
- Добавить расход:
- Получить статистику:

Бот выведет сумму доходов, расходов, прибыль и текущее количество студентов.
### Работа с Dashboard
После запуска Dashboard открывается по адресу: http://127.0.0.1:8050

Это позволяет легко:

- создавать резервные копии,
- редактировать данные вручную (если необходимо),
- переносить проект на любую машину.
